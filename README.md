# E-Commerce Cloud Data Analytics Pipeline (Automated)

An end-to-end cloud data engineering pipeline that automates data generation, cloud warehousing, and live BI visualization.

## 🛠️ Tech Stack & Architecture
* **Language:** Python 3.x (Data generation & pipeline engine)
* **Cloud Data Warehouse:** Snowflake (Secure data storage & SQL querying)
* **Automation:** Windows Task Scheduler (Daily production pipeline trigger at 9:00 PM)
* **Business Intelligence:** Tableau Public (Interactive sales performance dashboard)

## 🚀 How to Run Locally
1. Clone this repository.
2. Install dependencies: `pip install snowflake-connector-python`
3. Execute the script: `python load_data.py`
